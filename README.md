# Harper

Harper is a lightweight, browser‑based spelling and learning tool.  
This repository contains the public, static version of the project, designed to run on GitHub Pages.

## Features
- Clean HTML/CSS/JS structure
- Modular design
- Fully client‑side
- No server dependencies

## Notes
Private files (e.g., Harper.txt) are not included in this repository and remain local only.

## Hosting
This site is deployed using GitHub Pages.
